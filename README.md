# DirectX 12 Solutions

This repository contains my exercise solutions from *Introduction to 3D Game Programming with DirectX 12* 
by Frank Luna (2nd Edition). Every solution is my own implementation, 
sometimes with AI assistance to help me understand concepts more clearly.

## Notes

- These solution files contain `.cpp` and `.h` files only. If you want to run my solutions, 
you will need the framework from [Frank Luna's website](http://www.d3dcoder.net/d3d12_v2.htm).
